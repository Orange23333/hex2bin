# hex2bin

一个将16进制字符串转化成二进制文件。

## 命令行

### 安装

* 下载[`hex2bin.py`](./hex2bin.py)即可。

### 命令示例

* `python hex2bin.py out.bin direct "12345678ABCDEF"`
  参数依次为`输出路径`、`输入数据格式`、`输入数据`。

### 支持的输入格式

* `direct`: 连续的十二进制字符串。

## 其他

* 本脚本使用`Python 3.10.4`测试。
